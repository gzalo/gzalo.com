This site contains resources to practice for the amateur radio exam in Argentina.

## Available content

- **Morse Code (Theory)**: Fundamentals of Morse code for communications
- **Regulation practice**: Exercises on amateur radio rules and regulations
- **Technical theory, propagation and antennas practice**: Exercises on technical aspects of amateur radio
- List of **active repeaters** in Buenos Aires metropolitan area
