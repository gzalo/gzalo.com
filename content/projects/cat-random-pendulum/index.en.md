This weekend project is a simple cat toy consisting of a rod that moves randomly every few seconds. Instead of a solid rod, a cable or thread can also be used.

There's a demo video with real cats (Arwen y Bilbo): [Watch on YouTube](https://www.youtube.com/watch?v=83AkypU0tw0)

The source code is available at: [GitHub - cat-random-pendulum](https://github.com/gzalo/cat-random-pendulum)
