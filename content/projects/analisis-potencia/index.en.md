This project implements a side-channel attack on an embedded AES-128 implementation using Correlation Power Analysis (CPA).

This work was presented at SASE 2017 (Argentine Symposium on Embedded Systems).

The source code and more details are available at: [GitHub - analisis-potencia](https://github.com/gzalo/analisis-potencia)
