PDC32 is a post-apocalyptic computer built from scratch using recycled TTL chips by the awesome [Roberto Gudiño](https://twitter.com/computer_engin).

This project is a microinstruction-level emulator that allows running PDC32 programs in any web browser, without needing to install anything.

- [Web emulator](https://pdc32.github.io/pdc32-emu/) - Can be used directly in the browser
- [GitHub Repository](https://github.com/pdc32/pdc32-emu)
