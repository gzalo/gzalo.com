This project allows printing random images on Verifone VX520 point of sale terminals and similar devices.

It was created for retro technology events and fairs, where attendees can take home a small souvenir printed on thermal paper.

The source code is available at: [GitHub - posnet-image-printer](https://github.com/gzalo/posnet-image-printer)
