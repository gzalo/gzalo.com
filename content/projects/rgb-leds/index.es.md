Este proyecto permite controlar tiras LED RGB de 12V desde una PC mediante una interfaz web.

La interfaz web puede abrirse directamente en cualquier versión moderna de Chrome sin necesidad de un servidor.

El código fuente está disponible en: [GitHub - RGBLeds](https://github.com/gzalo/RGBLeds)
