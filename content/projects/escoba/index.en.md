The project consists of the design and implementation of a system to optimize the waste collection process through a network of wireless sensors installed in waste containers (using LoRa), which monitor and transmit their status to a central server. The server collects information from multiple containers and is responsible for planning the most efficient routes for collection vehicles, reducing the number of necessary collections.

Proyecto realizado junto a Ignacio Dorgan.
